# Quickstart: Interactive Digit Detector

**Date**: 2025-09-18

This guide provides the steps to test the interactive digit detector feature.

## Prerequisites
- The backend server is running.
- The frontend development server is running.

## Steps

1.  **Navigate to the 1950s Decade Page**
    - Open your web browser and go to the page for the 1950s decade.

2.  **Draw a Digit**
    - Locate the interactive digit detector section.
    - Use your mouse or touch screen to draw a single digit (0-9) on the canvas.

3.  **Predict the Digit**
    - Click the "Predict" button.

4.  **Verify the Prediction**
    - The application should display the digit that the AI model predicted.
    - For example, if you drew a "7", the application should display a message like "The AI predicts you drew a 7".

5.  **Clear the Canvas**
    - Click the "Clear" button to erase the canvas and draw a new digit.
